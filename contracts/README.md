# contracts

Contract documents for the instruction, selection, and confirmation flow that agents must follow.

- `INSTRUCTION_CONTRACT_V0.md`: minimum procedure for handling a JustPaste link (v0)
- `SELECTION_CONFIRMATION_FLOW_V0.md`: initial selection + confirmation flow
- `SELECTION_CONFIRMATION_FLOW_V1.md`: fixed first-apply flow for selection, refetch, scope, dry-run, and explicit approval
- `PRE_APPLY_CHECKLIST_V1.md`: checklist that must be shown again immediately before apply
