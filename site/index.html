JustPaste

링크만 붙여넣으면, 에이전트가 스킬을 세팅합니다

이 페이지는 사람도 읽을 수 있지만, 핵심은 에이전트가 지침을 읽고 바로 적용할 수 있도록 설계되어 있습니다.

중요: 레포 경로가 달라도 동작하도록 API는 상대경로 기준으로 제공합니다.

에이전트용 API 엔드포인트

./api/v1/packs/index.json (스킬 목록)
./api/v1/packs/commit-convention/instruction.json
./api/v1/packs/pr-clarity-template/instruction.json

배포 페이지에 명시된 스킬 내용

커밋 컨벤션 스킬
skill id: commit-convention

목적: 커밋 메시지를 컨벤셔널 포맷으로 통일해 협업 가독성을 높입니다.

  1. 현재 저장소의 기존 커밋 스타일을 확인합니다.
  2. 사용자에게 기본 형식 <type>(<scope>): <summary> 를 안내합니다.
  3. 적용 범위(프로젝트/전역)를 사용자에게 확인합니다.
  4. 기존 규칙 파일이 있으면 덮어쓰기 여부를 확인합니다.
  5. 반영 후 샘플 커밋 2개를 제시합니다.
PR 명확화 템플릿 스킬
skill id: pr-clarity-template

목적: PR 설명을 요약/범위/검증 중심으로 통일해 리뷰 속도를 높입니다.

  1. 기존 PR 템플릿 존재 여부를 확인합니다.
  2. 요약, 변경 내용, 범위, 검증, 관련 이슈 구조를 제시합니다.
  3. 적용 범위(현재 프로젝트/전역 기본값)를 확인합니다.
  4. 기존 템플릿 병합 또는 대체 여부를 확인합니다.
  5. 샘플 PR 본문 1개를 함께 제공합니다.
